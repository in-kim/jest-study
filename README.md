# Jest library Study

## Getting Started

```bash
npm install
#or
npm test
```

## Jest 라이브러리 설치

```bash
$ npm i -D jest
# 자동완성
$ npm i -D @types/jest
```

## test 스크립트 수정

```bash
"scripts": {
  "test": "jest"
},
```
